Oberheim Matrix 1000 Editor

Browser-based real-time parameter editor for the Oberheim Matrix 1000 synthesizer.

  • Sends SysEx parameter edits via Web MIDI API
  • Requires Chrome with MIDI+SysEx permission
  • Designed for V1.11 firmware
